# 🚗 Road Smart Aid - Помощник водителя

**Road Smart Aid** — это прогрессивное веб-приложение (PWA), которое помогает водителям получать быстрый доступ к информации о своих правах, обязанностях и правилах поведения на дороге.

## ✨ Основные возможности

- 📋 **Проверка прав водителя** — полная информация о необходимых документах и обязанностях
- 🚔 **Правила при остановке** — что делать при остановке инспектором ГИБДД
- ⚠️ **Процедуры при ДТП** — пошаговые инструкции при дорожно-транспортном происшествии
- 📱 **Работает оффлайн** — приложение кэширует данные для работы без интернета
- 🎨 **Адаптивный дизайн** — оптимизировано для всех устройств (мобильные, планшеты, ПК)
- 🌙 **Темный режим** — поддержка темного режима оформления

## 🚀 Быстрый старт

### Установка

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/dmitry7181-tech/road-smart-aid.git
cd road-smart-aid
```

2. **Откройте в браузере:**
   - Локально: откройте файл `index.html` в браузере
   - Или запустите локальный сервер:
```bash
python -m http.server 8000
```
Затем откройте `http://localhost:8000` в браузере.

### Установка как приложение

Road Smart Aid поддерживает установку как отдельное приложение:
1. Откройте приложение в браузере
2. Нажмите на кнопку "Установить" или "Добавить на главный экран"
3. Приложение будет установлено на ваше устройство

## 📋 Функции приложения

### 1. Проверка прав водителя
Информация о:
- Необходимых документах при управлении автомобилем
- Обязанностях водителя
- Правах водителя перед сотрудниками ГИБДД

### 2. Правила при остановке
Подробные инструкции:
- Как себя вести при остановке
- Какие требования инспектора законны
- Ваши права при остановке

### 3. Процедуры при ДТП
Пошаговые действия:
- Что делать сразу после ДТП
- Как документировать место происшествия
- Как оформить страховой случай

## 🔧 Технологический стек

- **HTML5** — семантическая разметка
- **CSS3** — адаптивный дизайн, темный режим
- **JavaScript (Vanilla)** — логика приложения без зависимостей
- **Service Worker** — оффлайн-функциональность
- **PWA** — установка как приложение

## 🔄 Автоматические обновления

Приложение включает GitHub Actions workflow, который:
- ⏰ Запускается каждый понедельник в 9:00 UTC
- 🔍 Проверяет изменения в официальных документах (Приказ МВД РФ № 264)
- 📝 Автоматически обновляет базу данных при обнаружении изменений
- 🔐 Использует хеширование для отслеживания изменений

## 📚 Источники информации

Приложение основано на следующих официальных документах:
- Приказ МВД России № 264
- Правила дорожного движения РФ
- Кодекс об административных правонарушениях РФ
- Бюллетень практики Верховного суда
